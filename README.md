# Viver de Renda App

Aplicativo Android desenvolvido para ajudar no planejamento, simulação de independência financeira e controle de investimentos.

## Executando Localmente

**Pré-requisitos:** [Android Studio](https://developer.android.com/studio)

1. Abra o Android Studio
2. Selecione **Open** e escolha o diretório contendo este projeto
3. Permita que o Android Studio corrija eventuais incompatibilidades ao importar o projeto (Gradle Sync)
4. Execute o aplicativo em um emulador ou dispositivo físico
